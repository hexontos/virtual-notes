# Defined Behaviour


Virtual Notes

A CLI POSIX app that stores virtual notes in a virtually defined place, but behaves as if the notes are in a local folder when the user creates some.

It will use a user-defined editor for notes.

In the terminal, you will use the tool like you would use standard Unix tools such as ls, but it also tracks notes that have a virtual path to the current folder, even though they are stored in a different place. Therefore, it doesn't create a binary mess in your folders.
